# Personal Portfolio Website

A premium, responsive personal portfolio website designed for a Senior Machine Learning Engineer and Data Scientist. This project showcases professional experience, skills, certifications, projects, and research publications with a modern, high-contrast aesthetic.

## Features

-   **Ultra-Premium Design**: A sophisticated UI with glassmorphism effects, refined typography, and a custom color palette.
-   **Dark/Light Mode**: Fully supported themes with a persistent toggle and smooth transitions.
-   **Interactive Elements**:
    -   Typing animation in the hero section.
    -   Smooth scroll navigation with active link highlighting.
    -   Fade-in scroll animations for all sections.
    -   Interactive skill bars and timeline cards.
    -   Subtle canvas-based background particle animation.
-   **Responsive Layout**: Optimized for all devices, from desktops to mobile phones.
-   **Sections**:
    -   **Home**: Hero section with profile card and summary.
    -   **About**: Detailed bio, stats, and industry experience tags.
    -   **Education**: Highlighted academic achievements with scholarship badges.
    -   **Experience**: Interactive vertical timeline of professional roles.
    -   **Skills**: Categorized technical skills with progress bars.
    -   **Certifications**: Grid of professional certifications with verification links.
    -   **Projects**: Featured projects with tech stacks and impact metrics.
    -   **Publications**: List of research papers with DOI/PDF links.
    -   **Contact**: Functional contact form (Formspree) and social links.

## Technologies Used

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom properties (variables), Flexbox, Grid, and Animations.
-   **JavaScript (ES6+)**: DOM manipulation, Intersection Observer, Canvas API.
-   **Fonts**: [Geist](https://vercel.com/font) and [Inter](https://fonts.google.com/specimen/Inter).
-   **Icons**: [Font Awesome](https://fontawesome.com/).

## Setup & Customization

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/awaismalik1x/portfolio.git
    ```
2.  **Open `index.html`** in your browser to view the site.
3.  **Customize Content**: Edit `index.html` to update your information.
4.  **Contact Form**: The form uses [Formspree](https://formspree.io/). Update the `action` attribute in the `<form>` tag with your own Formspree endpoint.
